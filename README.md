## Overview
Main describes the most stable version of the traditional Q-Learning approach, where each state is stored in its explicit board form. 
- This branch uses the StrictBoardRep knowledge base, which is easily identifiable by being slightly larger in size than the other
keyChangeTest changes state representation to describe the quality of the board and the value of performing a number of different game-altering moves.
- This branch uses the SemanticBoardRep knowledge base, which is slightly smaller than other file
